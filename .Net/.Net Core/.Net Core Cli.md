### 建立 git 版控 

安裝 dotnet-ignore 工具
```
dotnet tool update -g dotnet-ignore
```

建立 .gitignore 檔案，並建立 initial commit

```
dotnet ignore get -n VisualStudio
git init && git add . && git commit -m "Initial commit"
```
