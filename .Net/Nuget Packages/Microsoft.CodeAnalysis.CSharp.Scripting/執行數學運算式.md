# 執行數學運算式


```cs
using Microsoft.CodeAnalysis.CSharp.Scripting;
using Microsoft.CodeAnalysis.Scripting;

string expression = "((1*2) / (3*4*5)) + 6";

// 設定 Roslyn ScriptOptions
var options = ScriptOptions.Default
                            .WithImports("System")
                            .WithReferences(typeof(Program).Assembly);

// 使用 Roslyn CSharpScript.EvaluateAsync 方法來解析和運算表達式
var result = CSharpScript.EvaluateAsync(expression, options).Result;

Console.WriteLine(result); 
// 因為是 int 運算，所以 輸出結果 不會是 6.0333333333333332 而是 6
```
