# [建立測試專案](https://playwright.dev/dotnet/docs/intro)



```bash
# 建立方案資料夾
mkdir PlaywrightTests
cd PlaywrightTests

# 建立方案檔
dotnet new sln -n PlaywrightTests

# 建立專案
dotnet new nunit -n PlaywrightTests

# 加入專案至方案檔
dotnet sln add PlaywrightTests/PlaywrightTests.csproj

cd PlaywrightTests

# 安裝套件
dotnet add package Microsoft.Playwright.NUnit
dotnet build
pwsh bin/Debug/netX/playwright.ps1 install


# 建立測試檔案 (略)

# 錄制測試
pwsh bin/Debug/net7.0/playwright.ps1 codegen demo.playwright.dev/todomvc
" 等於
pwsh bin/Debug/net7.0/playwright.ps1 codegen https://demo.playwright.dev/todomvc
" 也可以不輸入 url 
pwsh bin/Debug/net7.0/playwright.ps1 codegen
```
