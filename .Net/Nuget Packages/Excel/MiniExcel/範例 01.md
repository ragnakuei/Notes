# 範例 01

Test.xlsx 示意

| A   | B   | C   |
| --- | --- | --- |
| 1   | 2   | 3   |
| 4   | 5   | 6   |

讀取

```cs
using MiniExcelLibs;

var filePath = @"C:\temp\Test.xlsx";
var dtos     = MiniExcel.Query<Dto>(filePath);

foreach(var dto in dtos)
{
    Console.WriteLine(dto.A);
}

public class Dto
{
    public string A { get; set; }

    public string B { get; set; }

    public string C { get; set; }
}

```
