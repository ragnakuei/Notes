# 一般函式

| 函式        | 功能                                 | 常用寫法           | 備註                                                |
| ----------- | ------------------------------------ | ------------------ | --------------------------------------------------- |
| RowNumber() | 取出該筆資料的 Row Number ( Base 1 ) | RowNumber(Nothing) | 如果用於群組判斷，只會顯示該 Group 最大的 RowNumber |
|             |                                      |                    |                                                     |

---

## 應用

- 每五筆一頁

  - 設定群組依據 `=Int((RowNumber(Nothing)-1)/5)`
  - 分頁符號 > 勾選`在群組的每個執行個體之間`
