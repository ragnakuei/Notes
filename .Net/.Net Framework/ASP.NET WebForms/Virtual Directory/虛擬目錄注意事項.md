# 虛擬目錄注意事項

1. js / css 的引用要用 ~/ 開頭 !

    ```html
    <link rel="stylesheet" href="~/lib/bootstrap/5.1.3/css/bootstrap.min.css" />
    <link rel="stylesheet" href="~/css/site.css" asp-append-version="true" />
    ```

1. aspx

    ```html
    <a href="<%# ResolveUrl("~/default.aspx") %>" />
    ```
