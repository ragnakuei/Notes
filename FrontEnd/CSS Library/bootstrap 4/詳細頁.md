# 詳細頁

## 最簡單做法

- 各欄位用 row 包起來
  - label 用 .col-form-label
    - 這樣就會有垂直對齊了 !
  - 輸入控制項用 div 包起來

### 範例

- 小於 sm 為 12
- 大於 sm 為 3 : 9

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-EVSTQN3/azprG1Anm3QDgpJLIm9Nao0Yz1ztcQTwFspd3yD65VohhpuuCOmLASjC" crossorigin="anonymous">

<div class="container">
    <div class="mb-3 row">
        <label for="exampleFormControlInput1" class="col-form-label col-12 col-sm-3">Account</label>
        <div class="col-12 col-sm-9">
            <input type="email" class="form-control" id="exampleFormControlInput1" placeholder="name@example.com">    
        </div>
    </div>
    <div class="mb-3 row">
        <label for="exampleFormControlInput1" class="col-form-label col-12 col-sm-3">Email</label>
        <div class="col-12 col-sm-9">
            <input type="email" class="form-control" id="exampleFormControlInput1" placeholder="name@example.com">    
        </div>
    </div>
    <div class="mb-3 row">
        <label for="exampleFormControlInput1" class="col-form-label col-12 col-sm-3">Name</label>
        <div class="col-12 col-sm-9">
            <input type="email" class="form-control" id="exampleFormControlInput1" placeholder="name@example.com">    
        </div> 
    </div>
</div>
```

## sm 以下，標題列靠左。sm 以上，標題列靠右

- sm (不含)以下
  - 標題列靠左
  - form-control 全滿   
- sm (含)以上
  - 標題列靠右
  - form-control 維持儘量一致 (各 break-point 都要給定固定比例)

### 做法二

- sm (不含)以下
  - 標題列靠左
  - form-control 全滿   
- sm (含)以上
  - 標題列靠右
  - form-control 維持儘量一致 (各 break-point 都要給定固定比例)
- .row 
  - sm (不含)以下
    - 將 inner html 的所有 dom 放在不同列
  - sm (含)以上
    - 將 inner html 的所有 dom 放在同一列

```html
<div class="row my-1">
    <dt class = "col-sm-2 text-left mx-3 text-sm-right ">
        <label for="Name"
                class="control-label text-nowrap">
            名稱
        </label>
    </dt>
    <dt class = "col-12 col-sm-6 col-md-5 col-lg-4 col-xl-3">
        <input id="Name"
                v-model="vm.Data.Name"
                class="form-control mx-3" />
    </dt>
</div>
<div class="row my-1">
    <dt class = "col-sm-2 text-left mx-3 text-sm-right ">
            <label for="FloatPrecision"
                    class="control-label text-nowrap">
                浮點數位數
            </label>
    </dt>
    <dt class = "col-12 col-sm-6 col-md-5 col-lg-4 col-xl-3">
            <input id="FloatPrecision"
                    type="number"
                    v-bind:step="minFloatPrecistion"
                    min="0"
                    max="0.1"
                    v-model="vm.FloatPrecision"
                    class="form-control mx-3" />
    </dt>
</div>
```
