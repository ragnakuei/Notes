### ISO-8601 格式

| ISO-8601                  | 說明                   |
| ------------------------- | ---------------------- |
| 2016-07-16T19:20:30+5:30  | timezone is ahead UTC  |
| 2016-07-16T19:20:30-01:00 | timezone is behind UTC |
| 2016-07-16T19:20:30Z      | UTC                    |
