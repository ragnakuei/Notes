# 範例 01

```html

<style>
    .test {
        width: 20px;
        height: 20px;
        background: blue;
        transition: all 1s;
    }
    
    .test.on {
        width: 200px;
        height: 200px;
    }
</style>

<div id="test" class="test"></div>

<button onclick="toggleOn()">Toggle</button>

<script>
    function toggleOn() {
        const block = document.getElementById('test');
        block.classList.toggle("on");    
    }
</script>
```
