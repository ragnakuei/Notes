# 範例 02

- 點一下，橙色為底，從中間展開
- 點一下，紅色為底，從二旁縮回

```html
<html lang="en">

<head>
    <meta charset="utf-8">
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/css/bootstrap.min.css" rel="stylesheet"
        integrity="sha384-EVSTQN3/azprG1Anm3QDgpJLIm9Nao0Yz1ztcQTwFspd3yD65VohhpuuCOmLASjC" crossorigin="anonymous">
</head>

<body>
    <style>
        .container {
            width: 100%;
        }

        .box {
            width: 5px;
            height: 0px;
            background: red;
            transition:
                /* step 1：duration*/
                height 1s,
                /* step 2：duration delay  */
                width 0.5s 0.5s;
        }

        .box.on {
            width: 300px;
            height: 100px;
            background: orange;
        }
    </style>

    <div class="container d-flex flex-column align-items-center">
        <div class="box m-2"></div>
        <div>
            <button class="btn btn-primary" onclick="toggleOn()">Toggle</button>
        </div>
    </div>

    <script>
        function toggleOn() {
            const targets = document.getElementsByClassName('box');
            for (const target of targets) {
                target.classList.toggle("on");
            }
        }
    </script>
</body>

</html>
```
