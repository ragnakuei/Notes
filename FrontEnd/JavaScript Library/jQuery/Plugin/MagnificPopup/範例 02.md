# 範例 02

單一視窗，手動開啟 / 關閉

```html
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/magnific-popup.js/1.1.0/magnific-popup.css" />
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/2.1.3/jquery.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/magnific-popup.js/1.1.0/jquery.magnific-popup.min.js"></script>
<style>
    .white-popup {
        position: relative;
        background: #FFF;
        padding: 20px;
        width: auto;
        max-width: 500px;
        margin: 20px auto;
    }
</style>

<div>
    <a href="#" onclick='openPopup()'>openPopup</a>
</div>

<div id="popup" class="white-popup mfp-hide">
    Popup content
    <div>
        <a href="#" onclick='closePopup()'>closePopup</a>
    </div>
</div>

<script>
    
    function openPopup() {
        $.magnificPopup.open({
            items: {
                src: '#popup',
                type: 'inline',
            }
        });
        window.popup = $.magnificPopup.instance;
    }

    function closePopup() {
        window.popup.close();
    }
</script>
```
