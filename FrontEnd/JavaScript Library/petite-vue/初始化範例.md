# 初始化範例

```html
<script src="https://unpkg.com/petite-vue" defer init></script>

<div v-scope="{ count: 0 }">
  {{ count }}
  <button @click="count++">inc</button>
</div>
```

少了 defet init，改用語法初始化


```html
<script src="https://unpkg.com/petite-vue"></script>

<div v-scope="{ count: 0 }">
  {{ count }}
  <button @click="count++">inc</button>
</div>

<script>
  PetiteVue.createApp().mount()
</script>
```

以 es module 初始化

```html
<div v-scope="{ count: 0 }">
  {{ count }}
  <button @click="count++">inc</button>
</div>

<script type="module">
  import { createApp } from 'https://unpkg.com/petite-vue?module'
  createApp().mount()
</script>
```

把 count 宣告放到 script 中

```html
<div v-scope>
    {{ count }}
    <button @click="count++">inc</button>
</div>

<script type="module">
    import { createApp } from 'https://unpkg.com/petite-vue?module'
    createApp({
        count: 0
    }).mount()
</script>
```

不使用 v-scope 改為指定 selector

```html
<div id="app">
    {{ count }}
    <button @click="count++">inc</button>
</div>

<script type="module">
    import { createApp } from 'https://unpkg.com/petite-vue?module'
    createApp({
        count: 0
    }).mount('#app')
</script>
```
