## 安裝 Vue Cli

> npm install -g @vue/cli@3.0.5

> vue --version

## 建立 vue 專案

> vue create [project name]

> project name 要全部小寫

## 執行開發階段 Server

> npm run serve

## 最小化編譯成 Production 環境

會將資料輸出至 dist 資料夾中

> num run build

## Lint 檢查

> npm run lint
