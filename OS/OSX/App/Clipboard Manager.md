# Clipboard Manager

## 免費又好用

### [Clipy](https://github.com/Clipy/Clipy)

點擊右方 Release 後，就可以看到下載連結

特殊設定：

1. 如果要按下 ctrl + command + v 直接出現可貼上的項目，而不是資料夾

    Preferences > Menu > Number of items placed inline > 設定大於 0 的數字就可以了 !
