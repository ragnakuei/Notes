# 調整 Result 視窗字型

- Edit > Preferences > Results > Style sheet for text(HTML) results > 選取 Custom > 
- 點擊 Lauch Editor
- 在 Customizations 輸入以下 css 語法

    ```css
    body
    {
        font-family: Monaco;
    }
    ```

- 完成
