重要設定

```json
// 設定 vimrc 的路徑
"vim.vimrc.path": "D:\\Kuei\\Notes\\Tools\\vim\\vimrc\\vscode.vimrc",

"vim.vimrc.enable": true,

// 行號顯示方式
"editor.lineNumbers": "relative",

// 是否要使用系統剪貼簿
"vim.useSystemClipboard": true,

// 是否要關閉 vim 的快捷鍵，以 vscode 的快捷鍵為主
"vim.handleKeys": {
    "<C-d>": false,
    "<C-c>": false,
    "<C-v>": false,
    "<C-s>": false,
    "<C-z>": false,
}
```
